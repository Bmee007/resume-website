Scroll-in Transitions — How should background images appear?

As the user scrolls into each section, how does the background image reveal itself?

A

Fade in

Image opacity animates from 0 → target (12–15%) over 0.8s. Clean, subtle, professional. Works uniformly across all sections.

opacity 0 → 0.15
B

Fade + subtle zoom

Image fades in while scaling from 1.05 → 1.0. Adds cinematic depth. Slight Ken Burns feel on entrance. Recommended for the "varied" look.

opacity 0→0.15 + scale 1.05→1
C

Varied per section

Different transition per section for maximum dynamism:

  • About → fade in (calm, bio-focused)
  • Experience → slide up + fade (timeline feel)
  • Projects → zoom in + fade (bold, dynamic)
  • Skills → fade in (clean grid)
  • Contact → fade in (minimal distraction)
